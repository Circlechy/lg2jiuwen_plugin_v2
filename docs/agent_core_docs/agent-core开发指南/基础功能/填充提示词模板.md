提示词模板往往包含可填充的占位符，占位符可以指代工具信息、示例、变量等内容。openJiuwen提供了提示词模板类`Template`，其中`content`表示模板内容，为必选，使用双花括号`{{...}}`标记需要后续填充的占位符，实际使用中将这些占位符替换为具体内容进行格式化，最终生成完整的提示词prompt。

# 实例化Template模板类

Template模板类支持以下三种类型`content`参数的输入，分别是`str`（字符串）、`List[Dict]`（字典列表）和`List[BaseMessage]`（`BaseMessage`列表）。以下介绍三种类型的`content`的实例化方式。

- `str`类型：实例化Template模板类，content类型为字符串类型。

  ```python
  from openjiuwen.core.utils.prompt.template.template import Template
  template = Template(content="你是一个精通{{domain}}领域的问答助手\n用户问题：{{query}}")
  ```

- `List[Dict]`类型：实例化Template模板类，content类型为字典列表类型。

  ```python
  from openjiuwen.core.utils.prompt.template.template import Template
  template = Template(
      content=[
          {"role": "system", "content": "你是一个精通{{domain}}领域的问答助手"},
          {"role": "user", "content": "用户问题：{{query}}"}
      ]
  )
  ```

- `List[BaseMessage]`类型：实例化Template模板类，content类型为BaseMessage列表类型。
  
  ```python
  from openjiuwen.core.utils.prompt.template.template import Template
  from openjiuwen.core.utils.llm.messages import SystemMessage, HumanMessage
  template = Template(
      content=[
          SystemMessage(role='system', content='你是一个精通{{domain}}领域的问答助手'),
          HumanMessage(role='user', content="用户问题：{{query}}")
      ]
  )
  ```

# 填充Template模板

实例化Template模板类后，可以调用`.format`方法对`content`进行填充，示例代码如下：

```python
messages = template.format({"domain": "数学", "query": "1+1=?"}).to_messages()
```
- 通过`str`类型生成的提示词模板，填充后结果为：

  ```python
  [
      HumanMessage(role='user', content='你是一个精通数学领域的问答助手\n用户问题：1+1=?', name=None)
  ]
  ```

- 通过`List[Dict]`类型生成的提示词模板，填充后结果为：

  ```python
  [
      SystemMessage(role='system', content='你是一个精通数学领域的问答助手', name=None),
      HumanMessage(role='user', content='用户问题：1+1=?', name=None)
  ]
  ```

- 通过`List[BaseMessage]`类型生成的提示词模板，填充后结果为：

  ```python
  [
      SystemMessage(role='system', content='你是一个精通数学领域的问答助手', name=None),
      HumanMessage(role='user', content='用户问题：1+1=?', name=None)
  ]
  ```

`{{...}}`标记的占位符会被实际值正确的替换，从而生成完整的输入内容。
