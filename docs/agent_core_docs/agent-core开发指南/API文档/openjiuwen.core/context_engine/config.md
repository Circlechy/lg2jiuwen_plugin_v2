# openjiuwen.core.context_engine.config

## class openjiuwen.core.context_engine.config.ContextEngineConfig
```python
class openjiuwen.core.context_engine.config.ContextEngineConfig()
```
表示上下文引擎的完整配置结构，用于控制上下文引擎的各种行为和参数。

**参数**：
- **conversation_history_length**(int, 可选)：对话历史记录长度，控制保存的对话消息数量。取值范围为大于等于0。默认值：100。
