# stream

`openjiuwen.core.stream`是支持openJiuwen实现流式输出功能的模块，它能够显著提升数据处理的实时性和效率。openJiuwen框架预置了多种流式处理模式，供用户直接使用。

**Classes**：

| CLASS                              | DESCRIPTION                                                          |
|------------------------------------|----------------------------------------------------------------------|
| [BaseStreamMode](./stream/base.md) | openJiuwen流式输出模式的枚举类。 |
| [OutputSchema](./stream/base.md)   | openJiuwen标准流式输出数据格式的数据类。 |
| [CustomSchema](./stream/base.md)   | 用户自定义流式输出数据格式的数据类。 |
| [TraceSchema](./stream/base.md)    | 调试信息流式输出数据格式的数据类。 |
